# Audience and Artifact Contract

## Intended Audience

This repository is written primarily for AI evaluation engineers, agent-safety researchers, technical reviewers, and teams assessing agentic AI behavior.

## Artifact Type

The repository contains:

- synthetic cases
- deterministic scorers
- generated reports
- regression tests
- narrow documentation of claims and limitations

## What It Is Not

It is not:

- a production safety certification
- a frontier-model evaluation result
- a complete agent-safety benchmark
- evidence about a private production system
- a substitute for independent evaluation

## Reviewer Question

The repository is designed to help reviewers ask:

> Does this synthetic agent response preserve authority boundaries and produce enough evidence for review under the tested conditions?
